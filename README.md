# Super Counter

A simple counter web app built with **Next.js** to practice the basics of React state, localStorage, and UI updates. This is a beginner-friendly project meant to learn by doing.

## ✨ Features

- Increment and decrement buttons
- Reset to zero
- Persistent count using localStorage
- State-driven UI updates
- Clean and responsive UI
- Built with Next.js

## 🛠️ Tech Stack

- Next.js
- React
- CSS Modules

## 🎯 Goal

Learn core front‑end concepts and get comfortable building small projects.
